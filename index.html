# Notes

最近学到的东西

05.01

CSS Grid 更适合处理页面骨架

侧栏、主内容、目录这类三栏结构，用 Grid 会比不断嵌套 Flex 更清晰。

04.24

搜索不一定要复杂

给内容块加上可检索关键词，再用 JavaScript 控制显示状态，就能做出轻量站内搜索。

04.18

作品描述要写清楚“为什么”

比起只写用了什么技术，更值得写清目标、取舍和自己负责的部分。
